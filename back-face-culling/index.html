Demonstrating back-face culling

How to explore this demo

Two files are important:

  1. js/main.js – this is the file that sets up the animation
  2. js/renderer.js – this is the file that does the actual 3d animation

You can (and probably should!) ignore the rest of the files. These two files are heavily commented so dive in for more information.

Libraries used by this demo

This demo uses 2 libraries:

  1. RequireJS (http://requirejs.org/)
    Really useful for writing nice, organised Javascript
  2. glMatrix (http://glmatrix.net/) Provides all the vector and matrix maths operations you could desire and works in a way that plays nicely with WebGL (should you want to explore that)
